Machine Learning Project

Crop Prediction

Problem Statement

About 50% of India's population depends on agriculture for their livelihood. Soil quality is one of the biggest factors that affects crop yield. Choosing the wrong crop for a given soil type degrades both the crop output and the soil itself. Over time, poor crop choices strip the soil of nutrients, creating a cycle of declining productivity.

Our Solution

We built a tool that checks soil conditions and recommends the best crop based on parameters like nutrient levels, moisture, and rainfall. The system is powered by a trained ML model that keeps recommendations as accurate as possible. Users enter their soil parameters and get an instant crop recommendation. It is simple to use and costs nothing to run.

Technical Stack

We trained a LightGBM classifier on a Kaggle dataset of soil and crop data. The frontend is HTML, CSS, and JavaScript. The backend runs on Python Flask. The trained model is served directly through the web interface, so users input their soil parameters and receive a prediction right away.
